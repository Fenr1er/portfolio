# Mein Portfolio

Dies ist mein persönliches Portfolio, welches ich als Referenz für zukünftige Arbeitgeber und zur Präsentation meiner Fähigkeiten und Projekte erstellt habe. Es wurde mit HTML5, CSS (Bootstrap) und JavaScript entwickelt, um eine responsive und interaktive Benutzererfahrung zu gewährleisten.

## Features

- **Responsive Design**: Das Portfolio ist vollständig responsive und funktioniert auf verschiedenen Geräten und Bildschirmgrößen, einschließlich der neuesten Modelle wie iPhone 15, iPhone 15 Pro, iPhone 15 Pro Max und Samsung S20.
- **Bootstrap**: Für das Styling wurde Bootstrap verwendet, was die Entwicklung beschleunigt und eine konsistente Optik gewährleistet.
- **JavaScript-Navigation**: Eine dynamische Navigation, die das Benutzererlebnis verbessert.
- **HTML5 Semantik**: Die Verwendung von HTML5-Semantik-Elementen sorgt für eine bessere Struktur und Zugänglichkeit.
